# Writing documentation

Writing style
---
- Headers/Titles use sentence case
- Acronyms are in **ALL CAPS**

Code style
---
- Use markdown variables for external links when possible so they are easy to fix in the future.
- Follow this style:

```md
# Article title

Header 1
---
Paragraph 1A

Paragraph 1B

Header 2
---
Paragraph 1B

Paragraph 2B
```
